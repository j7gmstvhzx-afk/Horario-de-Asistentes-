# Logo

Sube el archivo del logo de Casino Atlántico Manatí como
`logo-casino-atlantico.png` en este directorio `public/`.

Tamaño recomendado: 512×512 (PNG con fondo transparente) para que también
funcione como ícono de la PWA.

Mientras no exista el archivo, Next.js mostrará un error 404 para la imagen
— no rompe el resto de la app.
